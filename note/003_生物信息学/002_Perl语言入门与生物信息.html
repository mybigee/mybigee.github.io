前言

简介

  • Perl一般被称为实用报表提取语言（Practical Extraction and Report Language）。
  • Perl最初的设计者为拉里·沃尔(Larry Wall)他于1987年12月18日发表。
  • Perl大写的P，是指语言本身。
  • perl小写的p，是指程序运行的解释器。

Perl语言特点

  • 开源免费
  • 支持跨平台
  • 直接运行，无需复杂编译
  • 内部集成正则表达式
  • CPAN模块库

Perl语言的编程环境

  • Linux中自带perl语言的环境，输入perl -v查看Perl的版本。
  • Perl语言的扩展名为.pl

第一个程序

  • 使用vim创建一个新的Perl脚本文件，vim 001.pl

  • 第一行输入Perl的安装路径：#!/usr/bin/perl

    #!/usr/bin/perl
    #This is my first program
    print "Hello,world!\n";

  • 运行脚本程序：perl 文件名，perl 001.pl

vim的使用

  • 剪切光标所在的行：dd
  • 剪切光标到行首（第一个非空字符）：d^
  • 剪切光标到行尾：d$
  • 剪切光标后指定的行数（包括光标所在的行）：ndd，如3dd
  • 删除向上或向下的指定行数：dn，如d6，然后按向上键，表示向上删除6行（包括光标行是7行）。
  • 复制光标所在的行：yy，5yy表示向下复制5行。
  • 复制向上或向下的指定行数：yn，如y6，然后按向上键，表示向上复制6行（包括光标行是7行）。
  • 粘贴复制的内容：p
  • 回退：u
  • 回退多了前进：Ctrl + r
  • 跳到文件开头：[[
  • 跳到文件结尾：]]
  • 水平分屏：:split，退出使用:q
  • 垂直分屏：:vsplit，退出使用:q
  • 变量的补齐：Ctrl + N或Ctrl + P

处理生物文件实例

  • 处理blast的比对结果文件

  • 筛选出特征（identity）值大于50%，长度大于100bp的比对结果（行）

  • 创建Perl的脚本文件：filier_blast.pl

    ​x
    #!/usr/bin/perl -w 
    # -w 表示如果出错，着输出警告信息
    

    open IN,"blast_m8.out"; # 打开一个文件
    while (<IN>) {
            chomp; # 去掉行尾的换行符
            my @line=split /\s+/,$_; # 将读进来的每一行分割为列，identity为第3，长度为第4列
            if ($line[2] >= 50 && $line[3] >= 100) {
                    print "$_\n"; # 输出该行
            } else {
                    next; # 跳过该行
            }
    }
    close IN; # 关闭输入

split和join

  • split：将字符串根据固定的分隔符进行切割，切割后得到一个数组。第3个参数表示分割的个数。
  • join：作用相反，将数组连接成一个标量。
xxxxxxxxxx
#!/usr/bin/perl
$str = "123:abc:000";
@arr = split /:/,$str;
print "$arr[2]\n"; # 000
$new_str = join ":",@arr;
print "$new_str\n"; # 123:abc:000


$str2 = "123:333:ooo:kkk";
@arr2 = split /:/,$str2,3;
print "$arr2[2]\n"; # ooo:kkk

排序和翻转

  • sort：根据ASCII码进行排序，返回新的变量，原来的变量不变。
  • reverse：将原来的数组或字符串翻转，返回新的变量，原来的变量不变。
xxxxxxxxxx
#!/usr/bin/perl
@arr = (1..10);
@arr_new = sort(@arr);
print "@arr_new\n";
@arr_new = reverse sort(@arr);
print "@arr_new\n";
$dna = "ATCGGGAAT";
$and = reverse $dna;
print "$and\n";

#!/usr/bin/perl
use strict;
use Data::Dumper;

子程序

定义

  • 定义子程序使用sub关键字定义。
  • 调用子程序使用&加子程序名来调用，当子程序的名字与内置函数的名字不冲突的时候，可以不加&。
  • 类似于其他语言里的函数或方法。
xxxxxxxxxx
#!/usr/bin/perl
use strict;
&print_hello;
&print_hello;
sub print_hello {
        print "hello world\n";
}
xxxxxxxxxx
hello world
hello world

输入输出

  • 如果有return，会返回指定的值，如果没有子程序会将写在最后的表达式的结果作为返回值。return之后的语句不会执行。
  • 子程序的参数默认存储在@_数组中。
xxxxxxxxxx
sub sum{
        my $n1 = shift @_;
        my $n2 = shift @_;
        return $n1 + $n2;
}
print sum(3,4)."\t";
print &sum(4,5)."\t";
print sum(2)."\n";

7   9   2

实例：统计fasta信息

  • 10个基因序列文件（gene文件夹）和一个基因列表文件（gene.list）。
  • 统计这些文件的基因数目和基因的平均长度。
xxxxxxxxxx
#!/usr/bin/perl
use strict;


open IN,"<$ARGV[0]";
while (<IN>){
        chomp;
        # 基因名字与路径之间使用 = 隔开
        my ($geneName,$genePath) = (split /=/,$_)[0,1];
        my ($geneNum,$geneLength) = &geneInfo($genePath);
        print "$geneName\t$geneNum\t$geneLength\n";
}


# 接收一个基因文件的文件名，返回文件中的基因数目和平均长度。
sub geneInfo{
        my $path = shift @_;
        my $num = 0;
        my $length = 0;
        open FA,"<$path";
        while (<FA>){
                if(/^>/){
                        $num += 1;
                }else{
                        $length += length($_);
                }
        }
        close FA;
        my $avgLength = $length/$num;
        return $num,$avgLength;
}

变量

  • 全局变量：在整个程序的任何地方都可以使用，默认情况下为全局变量。
  • 私有变量：只能在语句块之内使用，即在{}内使用。使用关键字my创建私有变量。
  • use strict;是编译命令，在程序的编译器开始运行前进行的选项设置。该语句的作用是让程序有严格的规范。如定义变量没使用my，或使用my重复定义。

注意问题

  • 一行语句结束需要加分号。

  • 使用use strict之后，变量定义需要使用my。

  • 变量的作用域。

  • 检查程序是否有语法错误，使用perl -c 文件名

    xxxxxxxxxx
    [my@mycentos 19]$ perl -c test.pl

  • Perl的debug功能，使用perl -d 文件名，

    • 输入h显示帮助信息，
    • 输入l显示后10行，l 4-8，显示第4到第8行。
    • 输入q退出。
    • 输入n跳过子程序。
    • 输入b 数字设置断点。
    • 输入c程序继续执行，直到下一个断点或程序结尾。
    • 输入L查看程序的断点。
    • 输入R从头重新开始调试。
    • 输入s进入子程序。
    • 输入x输出引用与内部结构。

 10 $str =~ s/^/hahahaha/;
