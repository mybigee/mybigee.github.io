生物信息名词

生物信息名词英语中文名词物种常见的基因组版本区别

英语

  • WGAS（Genome-Wide Association Studies）：全基因组关联研究。
  • GSEA（Gene Set Enrichment Analysis）：基因探针富集分析。
  • TCGA（The Cancer Genome Atlas）：癌症基因组图谱。
  • SNP（Single Nucleotide Polymorphism）：单核苷酸多态性，主要是指在基因组水平上由单个核苷酸的变异所引起的DNA序列多态性。它是针对“群体”而言的，在群体中应占有一定的比例。
  • SNV（single nucleotide variants）：单核苷酸位点变异，没有频率限制，可用于描述任意一个可以被测序检测到的碱基突变。
  • reads（读长）：指的是测序仪单次测序所得到的碱基序列。
  • adapter（接头）：为一段已知的短核苷酸序列，用于链接未知的目标测序片段。
  • Insert（插入片段）：通俗解释就是两个Adapter接头中间的，被read的片段，即被打断的目标片段。
  • cluster：指二代测序所用芯片表面或单个磁珠表面生成的由单个DNA模板生成的数百至数千个DNA分子的集合。
  • Duplication Reads：指多个完全相同的DNA片段形成了多个有效cluster，读取这些Cluster所获得reads信息也是完全相同。
  • Base calling（碱基识别）：指的是从高通量测序仪（例如 Illumina、PacBio、Oxford Nanopore 等）生成的原始信号（Raw Signal）中提取出基因组序列的过程。
  • PE（Paired End）：成对末端末端测序、双端测序。
  • GSEA（Gene Set Enrichment Analysis）：基因集富集分析。
  • WGS（Whole Genome Sequencing）：全基因组测序。
  • DEG（Differentially Expressed Genes）：差异表达基因。表示两个或多个实验组之间在基因表达水平上存在显著差异的基因。
  • FDR（False Discovery Rate）：假阳性发现率，是一种多重假设检验校正方法。FDR通过控制假阳性发现率的方法，提供了一种确定性阈值的方案，从而保证在多个比较中发现的假阳性数量是可接受的。Q值可以解释为在显著性水平上，错将假阳性定义为真阳性的概率。一般来说，FDR 值越低，说明发现假阳性的概率越小，这通常被认为是更好的结果。
  • WGCNA（Weighted correlation network analysis）：加权共表达网络分析。
  • CAMERA（Consensus Analysis of Metabolite Profiles）：代谢产物图谱的一致性分析，是一种用于代谢组学数据分析的统计方法。它可以用于比较两类或多类不同条件下的代谢物谱图，并找到在这些条件之间显著变化的代谢物。
  • ROAST（Rotation Gene Set Tests）是一种常用的基因集差异分析方法，能够在基因表达数据中鉴定差异表达的代谢物组或基因集，以及它们与注释的生物学通路、功能、疾病等之间的关联性。其基本思想是将代谢物组表达模式旋转到一个新方向，使得旋转后的数据最大化基于代谢物组内基因的平均方差，并按照统计偏差来计算代谢物组的显著性。
  • TMM (trimmed mean of M-values) 标准化方法是一种在 RNA-seq 数据分析中常用的标准化方法，它可以有效地纠正样本间的测序深度差异和基因长度偏差。
  • GEO（Gene Expression Omnibus）：基因表达综合数据库。
  • WGCNA （Weighted gene correlation network analysis）：加权基因共表达网络分析，是描述不同样品之间基因关联模式的系统生物学方法。
  • SRA（Sequence ReadArchive）：数据库是NCBI用于存储二代测序的原始数据，包括 454，Illumina，SOLiD，IonTorrent等。
  • BED（Browser Extensible Data）：一种文本文件格式，用于存储基因组上的区域描述信息。
  • VCF (Variant Call Format)：用于描述SNP、INDEL和SV的结果的文本文件。
  • GFF（general feature format）：GFF文件是一种用来描述基因组特征的文件，现在我们所使用的大部分都是第三版（gff3）。
  • GTF（gene transfer format）：主要是用来对基因进行注释，当前所广泛使用的gtf格式为第二版（gtf2）。
  • GO（Gene Ontology）：一个用于描述基因和蛋白质功能的标准化分类体系，旨在统一描述所有生命形式中的基因和蛋白质的功能和相互关系。它是一种结构化的词汇，按照三个级别对生命过程中的基本结构、分子功能和生物学过程进行分类。http://geneontology.org/
  • KEGG（Kyoto Encyclopedia of Genes and Genomes）：一个综合性的数据库，它整合了生物化学、基因组学、代谢学和系统生物学等多个领域的数据和信息，并提供了一系列工具和资源，以帮助研究人员更好地理解生物和其调控机制。https://www.genome.jp/kegg/
  • reactome：一个用于生物学领域的免费开源数据库，提供了生物分子和化学反应的详细信息。Reactome 数据库主要关注代谢、信号传导和基因调控三个方面，并提供了包括人类、小鼠、大鼠、果蝇、斑马鱼、蛋白质等多种不同物种的数据。https://reactome.org/
  • MSigDB（Molecular Signatures Database）：一个基因集数据库，这些基因集包括来自文献报道的已知的代谢途径、信号通路、基因本体和其他生物过程的基因集，以及从基因组大规模数据中发现的基因集，例如转录因子结合位点和差异表达的基因集等。https://www.gsea-msigdb.org/gsea/msigdb

中文名词

  • 测序质量：

    • fastq第4行测序质量：对应ASCII字符减去33或64，数字越大，质量越好。
    • 测序质量与错误率（P）的测序质量的关系为：−10log10(P)

物种

  • Hs：指人（Homo sapiens），也称为人类；
  • Mm：指小鼠（Mus musculus），也称为家鼠；
  • Rn：指大鼠（Rattus norvegicus），也称为实验鼠；
  • Dm：指果蝇（Drosophila melanogaster），也称为飞蝇；
  • Pt：指斑马鱼（Danio rerio），也称为锦鲤鱼或斑马鱼。

常见的基因组版本

  • 人类（Homo sapiens）：GRCh38/hg38、GRCh37/hg19、NCBI36/hg18、HuRef
  • 小鼠（Mus musculus）：GRCm39/mm39、GRCm38/mm10、NCBIM37/mm9
  • 大鼠（Rattus norvegicus）：RGSC6.0/rn6、RGSC5.0/rn5
  • 果蝇（Drosophila melanogaster）：BDGP6/dm6、BDGP5/dm3
  • 斑马鱼（Danio rerio）：GRCz11/danRer11、GRCz10/danRer10、Zv9
  • 拟南芥（Arabidopsis thaliana）：TAIR10
  • 猪（Sus scrofa）：Sscrofa13.1/susScr11、Sscrofa11.1/susScr3、Sscrofa10.2/susScr2
  • 犬（Canis lupus familiaris）：CanFam3.1/canFam3、CanFam2.0/canFam2
  • 猴子（Macaca mulatta）：Mmul_10/Mmul_8.0.1、MGSCv3.1/rheMac3

区别

  • GRCh37 和 hg19 是同一个人类基因组版本的不同命名方式。
  • GRCh37 是由国际基因组组织（International Genome Organization）发布的该版本的名称，它是第二个完整的人类参考基因组版本，也是在科学界和医学界广泛使用的版本之一。
  • 而 hg19 则是由 UCSC（University of California, Santa Cruz）Genome Browser 发布的对该版本的命名，它是该浏览器所使用的第19个版本，并且是该浏览器上最常用的人类基因组版本之一。
  • 因此，GRCh37 和 hg19 都是用于描述同一个人类基因组版本，只是来源与用途不同，其命名方式也略有不同。
