二、NCBI中entrez ID的相关文件

  • 从 https://ftp.ncbi.nlm.nih.gov/gene/DATA/ 中与ID相关的文件：

  • 各个文件ID的解释文档：https://ftp.ncbi.nlm.nih.gov/gene/DATA/README

gene2go.gz文件

xxxxxxxxxx
# 读取文件，都只读取前100行数据
f <- "D:/桌面文件/文档/生物信息学/代码/008_ID转换/gene2go.gz"
goIDs <- read.table(f,sep = "\t",fill = T,nrow = 100)
names(goIDs) <- c("tax_id","GeneID","GO ID","Evidence",
                  "Qualifier","GO term","PubMed","Category")
  • 文件内容：

  • 各列的含义：

    • tax_id：NCBI Taxonomy的ID（物种ID）。
    • GeneID：NCBI Gene 数据库中基因的标识符，即entrez ID。
    • GO ID：GO的ID。
    • Evidence：在 gene_association 文件中，用于描述基因与 GO 词汇之间关联的数据源或证据类型。
    • Qualifier：进一步描述基因与 GO 词汇关系的限定词。
    • GO term：由 GO ID 所指示的 GO 术语名称和描述。
    • PubMed：一个有序的 PubMed UID 列表，这些 UID 与与基因和 GO 词汇之间的关联有关。
    • Category：GO 词汇所属的分类（即 Function、Process 或 Component）。

gene2ensembl.gz文件

xxxxxxxxxx
f <- "D:/桌面文件/文档/生物信息学/代码/008_ID转换/gene2ensembl.gz"
ensemblIDs <- read.table(f,sep = "\t",fill = T,nrow = 100)
names(ensemblIDs) <- c("tax_id","GeneID","Ensembl_gene_identifier",
                       "RNA nucleotide accession.version","Ensembl_rna_identifier",
                       "protein accession.version","Ensembl_protein_identifier")

  • 各列的含义：

    • tax_id：NCBI Taxonomy的ID，即物种ID。
    • GeneID：NCBI Gene 数据库中基因的标识符，即entrez ID。
    • Ensembl_gene_identifier：匹配的 Ensembl 基因标识符，即基因的Ensembl ID。
    • RNA nucleotide accession.version：与 NCBI RefSeq RNA 序列匹配的标识符。
    • Ensembl_rna_identifier：与 Ensembl RNA 序列匹配的标识符，即RNA的Ensembl ID。。
    • protein accession.version：与 NCBI RefSeq 蛋白质序列匹配的标识符。
    • Ensembl_protein_identifier：与 Ensembl 蛋白质序列匹配的标识符，即蛋白质的Ensembl ID。。

gene2accession.gz文件

xxxxxxxxxx
f <- "C:/Users/SYWS/Desktop/gene2accession.gz"
accessionIDs <- read.table(f,sep = "\t",fill = T,nrow = 100)

  • 各列的含义：

    • tax_id：NCBI Taxonomy的ID，即物种ID。
    • GeneID：NCBI Gene 数据库中基因的标识符，即entrez ID。
    • status：RefSeq 序列的状态。
    • RNA nucleotide accession.version：与 NCBI RefSeq RNA 序列匹配的标识符。
    • RNA nucleotide gi：RNA 序列的 GI 号。
    • protein accession.version：与 NCBI RefSeq 蛋白质序列匹配的标识符。
    • protein gi：蛋白质序列的 GI 号。
    • genomic nucleotide accession.version：基因组 DNA 序列的 Accession 和版本号。
    • genomic nucleotide gi：基因组 DNA 序列的 GI 号。
    • start position on the genomic accession：该基因在基因组 DNA 序列上的起始位置（0-based）。
    • end position on the genomic accession：该基因在基因组 DNA 序列上的终止位置（0-based）。
    • orientation：该基因在基因组 DNA 序列上的方向，可能值为 "+"、"-" 或 "?"。
    • assembly：该序列所属的组装名称。
    • mature peptide accession.version：成熟肽序列的 Accession 和版本号。
    • mature peptide gi：成熟肽序列的 GI 号。
    • Symbol：该基因的默认符号。

三、ID转换

  • 方法一：使用merge()函数，使用类似于数据集连接的方式进行ID转换。
  • 方法二：找到对应关系，然后使用match()函数。

方法一

xxxxxxxxxx
# 读取readsCounts.xlsx文件的第一列（基因ID），进行ID转换
library(openxlsx)
f <- "D:/桌面文件/文档/B站生信视频/代码/004_生信人应该这样学R语言/readsCounts.xlsx"
# colNames = T：存在表头，类似于header=T
# 只读取第一列cols = 1
geneID <- read.xlsx(f,sheet = "Sheet1",colNames = T,cols = 1)
names(geneID) <- "ensembl_id"
# 读取ID转换的包
library(org.Hs.eg.db)
# 转为数据框的格式
g2s <- toTable(org.Hs.egSYMBOL)
g2e <- toTable(org.Hs.egENSEMBL)
# 根据ensembl_id合并数据框
eids <- merge(geneID,g2e,by = "ensembl_id",all.x = T)
# 根据gene_id合并数据框
genes <- merge(eids,g2s,by = "gene_id",all.x = T)
# 排序
genes <- genes[order(genes$ensembl_id),]
# 原因是一个ensembl_id对应到了两个gene_id
# ensembl_id出现多于一次的基因
x <- table(genes$ensembl_id)[table(genes$ensembl_id) > 1]
# 去掉重复的行（将该行删除）
genes <- genes[!duplicated(genes$ensembl_id),]
# 根据原geneID的顺序排列
genes <- genes[match(geneID$ensembl_id,genes$ensembl_id),]
# 保存为csv
write.csv(genes,"d:/genes.csv")

affymetrix芯片探针数据转换

# affymetrix芯片
library("hgu95av2.db")
# 包里包含的数据集
ls('package:hgu95av2.db')
# 探针和entrez ID
probe2entrezID <- toTable(hgu95av2ENTREZID)
# 探针和symbol
probe2symbol <- toTable(hgu95av2SYMBOL)
# 探针和genename
probe2genename <- toTable(hgu95av2GENENAME)
head(probe2entrezID,2)
head(probe2symbol,2)
head(probe2genename,2)


# 随机抽取探针数据
probes = sample(unique(mappedLkeys(hgu95av2ENTREZID)),10)
# 添加冗余探针
probes[c(5,7)] <- probes[c(8,10)]
# 添加以上数据框不存在的探针
probes[6] <- "aaaa"
probes[11] <- "bbbb"
probes


# 以下分别用于获取探针的entrez ID、symbol、genename
index1 <- match(probes,probe2entrezID$probe_id)
index2 <- na.omit(index1) # 删除probe2entrezID中没有的探针
temp1 <- probe2entrezID[index2,]


temp2 <- probe2symbol[na.omit(match(probes,probe2symbol$probe_id)),]
temp3 <- probe2genename[na.omit(match(probes,probe2genename$probe_id)),]
