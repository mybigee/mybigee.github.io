标准的预处理工作流程

  • 下面的步骤包含在Seurat中的标准的预处理scRNA-seq数据的工作流程。
  • 基于QC指标选择和过滤细胞，数据标准化和归一化，寻找高可变的特征（feature）

QC和细胞选择

常用的QC指标

  • 检测每个细胞中唯一基因的数量

    • 低质量或空的液滴中的基因数量非常少
    • 两个或多个细胞会存在异常高的基因数量

  • 在一个细胞内检测到的分子总数（与一个细胞的唯一基因密切相关，即下面的nCount_RNA）

  • 比对到线粒体基因组上的read数

    • 低质量或死细胞中经常存在广泛的线粒体污染
    • 从一个特征基因集（features）计算计数的百分比为QC指标
    • 使用有MT-的开始的基因集作为线粒体基因

识别高可变基因（特征选择）

  • 计算出一个在细胞与细胞之间存在高变异的特征基因集（如部分基因在这些细胞高表达而在另一些细胞低表达）。关注这些基因有助于下游分析去标出生物学信号。
  • 使用均值-方差关系来改进之前的版本，默认返回2000个特征基因。
  • 使用FindVariableFeatures()
xxxxxxxxxx
# 根据基因的方差稳定性（Variance Stabilizing Transformation，VST）来选择具有最高变异性的基因
pbmc <- FindVariableFeatures(pbmc,selection.method = "vst",nfeatures = 2000)
# 变异最大的10个基因

数据归一化

  • 使用线性变换的方法，是降维（如PCA）之前的步骤，使用ScaleData()

    • 转换每个基因的表达量，使得跨细胞之间的平均表达量为0
    • 归一化每个基因的表达量，使得跨细胞的方差为1（在下游的分析中，每个细胞的权重相同，不会导致高表达的细胞主导数据）

  • 结果存储在pbmc[["RNA"]]@scale.data中。

细胞聚类

  • 在PCA空间中，基于欧氏距离创建一个KNN的图，同时优化任意两个细胞之间局部邻域中的共享重叠（Jaccard相似性）的边的权重，使用前面确定的10个PC和FindNeighbors()函数。
  • 为了对细胞进行聚类，使用模块优化技术，如Louvain（默认）、SLM。使用FindClusters()实现，同时使用resolution参数，设置下游聚类的“粒度”，值增加会导致聚类数增加。对于大约3千个细胞，将值设置在0.4-1.2之间的效果较好，随着数据集的增大，优化参数增大。
  • 聚类可以使用Idents()函数查看。
xxxxxxxxxx
# 发现邻居节点
pbmc <- FindNeighbors(pbmc,dims = 1:10)
# 细胞聚类
pbmc <- FindClusters(pbmc, resolution = 0.5)

寻找差异表达的feature

  • Seurat 可以通过差异表达，找出聚类的marker。默认情况下，找出一个细胞聚类的积极和消极的marker（通过ident.1参数指定细胞聚类）。FindAllMarkers()自动执行该过程，也可以指定各个细胞聚类之间的比较。
  • min.pct参数要求，被检测的feature在两组细胞中的任意一个有最小的百分比（表示至少在指定的细胞群中有多少百分比的细胞上基因要表达）。thresh.test参数要求feature在两组有一定的平均表达量的差异。可以将这两个参数都设置为0，但会显著增加运行的时间，因为会对许多没有区分的feature进行参数。
  • max.cells.per.ident参数的设置，可以提高运行的速度，通过下采样确定类，使其单元格数不超过其设置的单元格数。虽然通常会有功率损失，但速度提升可能显著。
xxxxxxxxxx
# 寻找差异表达的feature
