概述

  • 使用Seurat分析空间转录组的数据，分析的过程与单细胞转录组（scRNA-seq）的分析过程类似。

  • 空间分析的常用步骤：

    • 标准化（数据的预处理）
    • 降维和聚类
    • 探索空间的可变特征（feature）
    • 交互式可视化
    • 与单细胞数据的整合
    • 处理多个切片（对生物样本的切割）

  • 测序方法：10x Genomics的Visium技术、Slide-seq等

数据集

  • 矢状体小鼠脑切片数据集，有两个连续的前部切片和两个连续的后部切片。
  • 数据是spaceranger软件（去除低质量的测序读段、将转录本识别到细胞、生成一个包含各个基因在每个细胞中表达水平的矩阵等）的输出。
  • 数据使用Load10X_Spatial()函数加载，返回一个Seurat对象，该对象包含spot水平的表达数据以及组织切片的相关图像。
  • spot的大小可以看作是分辨率的大小，较小的Spot可以提供更高的空间分辨率，能够更准确地确定细胞核位置和基因表达的空间分布信息。
  • 数据的加载也可以使用SeuratData包。
  • 直接使用R语言下载数据较慢，容易出错，数据地址：http://seurat.nygenome.org/src/contrib/stxBrain.SeuratData_0.1.1.tar.gz，直接下载文件，本地安装。

数据预处理

  • 与scRNA-seq类似。
  • 第一步，标准化不同数据点之间由于测序深度所引起的差异。每个点的分子计数可以是有大的不同，特别是如果整个组织的细胞密度存在差异。
xxxxxxxxxx
# 绘制小提counts分布琴图
plot1 <- VlnPlot(brain, features = "nCount_Spatial", pt.size = 0.1) + NoLegend()
# 绘制空间特征图
plot2 <- SpatialFeaturePlot(brain, features = "nCount_Spatial") + theme(legend.position = "right")
wrap_plots(plot1, plot2)  # 并列显示

  • 差异的来源（spot的不同分子计数）：技术、组织解剖结构。例如，在神经元耗尽的组织区域，可表现出低的分子计数。因此，对于标准化的方法（LogNormalize()）使每个数据点有相同的大小会导致一些问题。
  • 因此使用sctransform的标准化的方法，它建立了基因表达的正则化负二项式模型，以便在保留生物变异的同时解释技术引起的误差。该方法的更多细节：https://doi.org/10.1186/s13059-019-1874-1。
xxxxxxxxxx

空间可变特征的识别

  • 两种方法：

    • 基于组织内预先注释的解剖区域进行差异表达，可以通过无监督聚类或先验知识来确定。
    • 没有预先注释，搜索表现出空间模式的特征，该方法将空间转录组学数据建模为标记点过程并计算"变异函数"，用于识别表达水平取决于其空间位置的基因

使用 RCTD 的空间反卷积

  • RCTD（Robust Cell Type Decomposition）稳健的细胞类型分解，一种在提供 scRNA-seq 参考时从空间数据集中解卷积spot水平数据的计算方法。可以准确地注释来自各种技术的空间数据。
xxxxxxxxxx
# 下载
devtools::install_github("dmcable/spacexr", build_vignettes = FALSE)
xxxxxxxxxx
library(spacexr)


# scRNA-seq的参考数据
ref <- readRDS("数据/mouse_hippocampus_reference.rds")
ref <- UpdateSeuratObject(ref)
Idents(ref) <- "celltype"


# 提取RCTD所需的数据
# 构造reference对象
counts <- ref[["RNA"]]$counts
cluster <- as.factor(ref$celltype)
names(cluster) <- colnames(ref)
nUMI <- ref$nCount_RNA
names(nUMI) <- colnames(ref)
reference <- Reference(counts, cluster, nUMI)


# 构造query对象
slide.seq <- SeuratData::LoadData("ssHippo")
counts <- slide.seq[["Spatial"]]$counts
coords <- GetTissueCoordinates(slide.seq)
colnames(coords) <- c("x", "y")
coords[is.na(colnames(coords))] <- NULL
query <- SpatialRNA(coords, counts, colSums(counts))
