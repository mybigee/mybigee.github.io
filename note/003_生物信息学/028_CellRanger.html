Run Summary

  • 程序运行的汇总信息（与运行的命令相关）。

Cells

  • Estimated Number of Cells

    • The number of barcodes associated with at least one cell.
    • 至少和一个细胞相关的barcode的数量。
    • 本次样本测到的细胞数，和实验上机的细胞数量和细胞质量有直接关系

  • Fraction Reads in Cells

    • The fraction of valid-barcode, confidently-mapped reads with cell-associated barcodes.
    • 有效barcode的分数，与细胞相关的barcode的可信匹配的reads。
    • 每个样本过滤处理后，细胞reads数占总reads数的百分比，表示与细胞相关的UMI可靠地比对到基因组，理想值是＞70%。如果样品中游离的mRNA很多，该数值相对就会偏低，这个参数反映测序数据的利用率。

  • Mean Reads per Cell

    • The total number of sequenced reads divided by the number of barcodes associated with cell-containing partitions.
    • 测序的总的reads数除以和细胞包含部分的barcode的数量。
    • 每个细胞测到的平均reads，一般是Number of Reads 除以Estimated Number of Cells的数值，至少测到20K reads才能反映真实的细胞状况，比较好的数据推荐测到30k reads/cell以上

  • Median UMI Counts per Cell

    • The median number of UMI counts per %s cell-associated barcode.
    • 与细胞相应的barcode的UMI数量的中位数。
    • 代表的是每个细胞中被检测到UMI数据的中位数。UMI是目前许多高通量单细胞测序平台用到的一种分子标签，会给细胞中每个被捕获的mRNA分子打上一个独特的标签，用来在分析中校准基因的表达量。通过这个指标，我们可以了解到每个高质量细胞中大概有多少个mRNA分子被捕获到。

  • Median Genes per Cell

    • The median number of genes detected per cell-associated barcode. Detection is defined as the presence of at least 1 UMI count.
    • 每个与细胞相关的barcode检测到的基因数量的中位数。检测定义为至少存在一个UMI的计数。
    • 每个细胞检测到的基因中位数。

  • Total Genes Detected

    • The number of genes with at least one UMI count in any cell.
    • 任一细胞中至少有一个UMI计数的基因的数量。
    • 检测到的基因总数，每个基因至少检测到一条UMI

  • Barcode Rank Plot

    • The plot shows the count of filtered UMIs mapped to each barcode. Barcodes are not determined to be cell-associated strictly based on their UMI count. Instead, they could be determined based on their expression profile, or removed via Protein Aggregate Detection and Filtering and/or High Occupancy GEM Filtering. Therefore, some regions of the graph contain both cell-associated and background-associated barcodes. The color of the graph in these regions is based on the local density of barcodes that are cell-associated. Hovering over the plot displays the total number and percentage of barcodes in that region called as cells along with the number of UMI counts for those barcodes and barcode rank, ordered in descending order of UMI counts.
    • 图像展示的是匹配到每个barcode的过滤后的UMI的计数。barcode不是严格根据它UMI的计数确定它和细胞相关。相反，它们可以根据它们的表达谱来确定，或者通过蛋白质聚集体检测和过滤和/或高占用率GEM过滤去除。因此，图形的一些区域包含与细胞和背景相关的barcode。在这些区域中的图形颜色是基于与细胞相关的barcode的局部密度。悬浮在图像上可以显示被称为细胞的区域的barcode的总数和百分比，以及这些barcode的UMI计数和排序。排序按UMI计数降序排列。

Sequencing

  • Number of Reads

    • Total number of read pairs that were assigned to this library in demultiplexing.
    • 在拆分过程中分配给这个文库（样本）的总的read pairs数。
    • 整个样本测到的reads总数

  • Number of Short Reads Skipped

    • Total number of read pairs that were ignored by the pipeline because they do not satisfy the minimum length requirements (for example read1 less that 26 bases in 3' v2 or 3' v3 or 5').
    • 由于不满足最小的长度要求而在流程中被忽略的read pairs数。

  • Valid Barcodes

    • Fraction of reads with barcodes that match the whitelist after barcode correction.
    • 与barcode校正之后的白名单匹配的barcode的read分数。
    • 带有正确10x Barcode的read的比例，每个Bracode对应到每个细胞。

  • Valid UMIs

    • Fraction of reads with valid UMIs; i.e. UMI sequences that do not contain Ns and that are not homopolymers.
    • 有效UMI的reads的分数。也就是不含有N和聚合物的UMI序列。

  • Sequencing Saturation

    • The fraction of reads originating from an already-observed UMI. This is a function of library complexity and sequencing depth. More specifically, this is the fraction of confidently mapped, valid cell-barcode, valid UMI reads that had a non-unique (cell-barcode, UMI, gene). This metric was called 'cDNA PCR Duplication' in versions of Cell Ranger prior to 1.2.
    • 来源于已经检测到的UMI的reads的分数。这是一个文库复杂度和测序深度的功能。具体来说，这个是一个可信的比对分数，有效的细胞barcode，由非唯一的有效的UMI reads（细胞barcode，UMI，基因）。在1.2之前的Cell Ranger版本中，这个指标被称为“cDNA PCR重复”。
    • 测序饱和度，一般60-80%左右，这个指标非常重要，如果捕获的细胞数较多，但是每个细胞里面的平均reads数少，那么饱和度就较低，反之饱和度较高。总的来说，测序饱和度越高，再增加测序量的意义就越小

  • Q30 Bases in Barcode

    • Fraction of cell barcode bases with Q-score >= 30, excluding very low quality/no-call (Q <= 2) bases from the denominator.
    • 碱基质量大于等于30的细胞barcode的碱基的分数，分母不包括质量非常低的（Q<=2）或没有识别的（no-call）的碱基。
    • Barcode序列中质量分数大于等于 30 的碱基百分比

  • Q30 Bases in RNA Read

    • Fraction of RNA read bases with Q-score >= 30, excluding very low quality/no-call (Q <= 2) bases from the denominator. This is Read 1 for the Single Cell 3' v1 chemistry and Single Cell 5' paired end, Read 2 for the Single Cell 3' v2/v3 chemistry and Single Cell 5' R2-only).
    • 碱基质量大于等于30的RNA read的碱基的分数，分母不包括质量非常低的（Q<=2）或没有识别的（no-call）的碱基。这是单细胞3'v1化学和单细胞5'配对端的read1，单细胞3'v2/v3化学和单细胞5'R2的read2。
    • 用于比对的Reads中质量分数大于等于 30 的碱基百分比。

  • Q30 Bases in UMI

    • Fraction of UMI bases with Q-score >= 30, excluding very low quality/no-call (Q <= 2) bases from the denominator.
    • 碱基质量大于等于30的UMI的碱基的分数，分母不包括质量非常低的（Q<=2）或没有识别的（no-call）的碱基。
    • UMI 序列中质量分数大于等于 30 的碱基百分比。

Mapping

  • Reads Mapped to Genome

    • Fraction of reads that mapped to the genome.
    • 比对到参考基因组的reads的分数。

  • Reads Mapped Confidently to Genome

    • Fraction of reads that mapped uniquely to the genome. If a gene mapped to exonic loci from a single gene and also to non-exonic loci, it is considered uniquely mapped to one of the exonic loci.
    • 唯一比对到参考基因组的read的分数。如果一个基因是比对到单个基因的外显子位点，同时也比对到非外显子位点，那么这个基因被认为是唯一的比对到外显子的一个位点。
    • 比对到参考基因组并得到转录本GTF信息支持的Reads占总Reads中占的比例；如果一条Reads既可以比对到一个基因的外显子区（exon区），又可以比对到非外显子区（非exon区），那么算Reads比对到外显子（exon区）。

  • Reads Mapped Confidently to Intergenic Regions

    • Fraction of reads that mapped uniquely to an intergenic region of the genome.
    • 唯一比对到参考基因组基因间区的read的分数。

  • Reads Mapped Confidently to Intronic Regions

    • Fraction of reads that mapped uniquely to an intronic region of the genome.
    • 唯一比对到参考基因组内含子区域的read的分数。

  • Reads Mapped Confidently to Exonic Regions

    • Fraction of reads that mapped uniquely to an exonic region of the genome.
    • 唯一比对到参考基因组外显子区域的read的分数。

  • Reads Mapped Confidently to Transcriptome

    • Fraction of reads that mapped to a unique gene in the transcriptome. The read must be consistent with annotated splice junctions when include-introns=false. These reads are considered for UMI counting.
    • 比对到转录本唯一基因的read的分数。当include-introns=false的时候，read必须是与带有剪切连接的一致。这些read被考虑用于UMI计数。
    • 比对到已知参考转录本的Reads在总Reads中占的比例；这一部分的Reads主要是对UMI进行计数统计。

  • Reads Mapped Antisense to Gene

    • Fraction of reads confidently mapped to the transcriptome, but on the opposite strand of their annotated gene. A read is counted as antisense if it has any alignments that are consistent with an exon of a transcript but antisense to it, and has no sense alignments.
    • 可靠的比对到转录组的reads的分数，但是位于注释基因的反义链上。如果一个read有任何比对和一个转录本外显子的反义一致，同时没有有意义的比对，那么就被当作是反义的。
    • 比对到基因反义链上的Reads所占的比例。

Top Features by Cluster (Log2 fold-change, p-value)

  • The differential expression analysis seeks to find, for each cluster, features that are more highly expressed in that cluster relative to the rest of the sample. Here a differential expression test was performed between each cluster and the rest of the sample for each feature. The Log2 fold-change (L2FC) is an estimate of the log2 ratio of expression in a cluster to that in all other cells. A value of 1.0 indicates 2-fold greater expression in the cluster of interest. The p-value is a measure of the statistical significance of the expression difference and is based on a negative binomial test. The p-value reported here has been adjusted for multiple testing via the Benjamini-Hochberg procedure. In this table you can click on a column to sort by that value. Also, in this table features were filtered by (Mean UMI counts > 1.0) and the top N features by L2FC for each cluster were retained. Features with L2FC < 0 or adjusted p-value >= 0.10 were grayed out. The number of top features shown per cluster, N, is set to limit the number of table entries shown to 10,000; N=10,000/K^2 where K is the number of clusters. N can range from 1 to 50. For the full table, please refer to the 'differential_expression.csv' files produced by the pipeline.

  • 差异表达分析旨在为每个聚类找到相对于其余样本在该聚类中表达更高的特征。这里对每个特征在每个聚类和其余样本样本之间进行了差异表达检验。Log2倍数变化（L2FC）是簇中表达与所有其他细胞中表达的Log2比率的估计。值1.0表示感兴趣簇中的表达增加了2倍。p值是表达式差异统计显著性的度量，基于负二项式检验。此处报告的p值已通过Benjamini-Hochberg程序进行了多重校正调整。在此表中，您可以单击列以按该值排序。此外，在该表中，特征按（平均UMI计数>1.0）进行过滤，并保留了每个聚类的L2FC前N个特征。L2FC<0或调整后的p值>=0.10的特征显示为灰色。每个集群显示的顶部特征数量N被设置为将显示的表条目数限制在10000；N=10000/K^2，其中K是簇的数量。N的范围可以从1到50。有关完整表格，请参阅流程生成的“difficulty_expression.csv”文件。

Sequencing Saturation

  • This plot shows the Sequencing Saturation metric as a function of downsampled sequencing depth (measured in mean reads per cell), up to the observed sequencing depth. Sequencing Saturation is a measure of the observed library complexity, and approaches 1.0 (100%) when all converted mRNA transcripts have been sequenced. The slope of the curve near the endpoint can be interpreted as an upper bound to the benefit to be gained from increasing the sequencing depth beyond this point. The dotted line is drawn at a value reasonably approximating the saturation point.

  • 该图显示了测序饱和度度量与下采样测序深度（以每个细胞的平均读数测量）的函数关系，直至观察到的测序深度。测序饱和度是衡量观察到的文库复杂性的指标，当所有转换的mRNA转录物都已测序时，饱和度接近1.0（100%）。终点附近的曲线斜率可以解释为将测序深度增加到该点以上所获得的益处的上限。虚线以合理近似饱和点的值绘制。
